## NON VÉRIFIABLE EN CODE-PREP (à exécuter avec DB / compilateur)

- **Migrations** `20260628_*` : `yarn payload migrate` sur copie Neon + idempotence (2×) + rollback (`MIGRATION.md §8`).
- **Index** GiST (`reseauteurs`/`evenements` geom) et GIN trigram (`pg_trgm`) : `EXPLAIN ANALYZE`.
- **Webhook Stripe** : `checkout.session.completed` → `evenement.premium=true` ; `customer.subscription.deleted` → `reseau.partenaire=false`.
- **JSON-LD** : validation Rich Results (`Person`/`Event`/`Organization`) + non-régression de l'échappement XSS.
- **noindex par défaut** : profil `en_attente` → `<meta robots noindex>` présent.
- **Slug collision déterministe** : 2 homonymes → `-<ville>`/`-2`, jamais `Date.now()`.
- **`generate:types`** puis lever les `@ts-nocheck` (`Reseauteurs.ts`, `Reseaux.ts`, composants carte) **et** supprimer dans la foulée : I-10 (builders JSON-LD legacy + imports de types legacy de `jsonld.ts`), M-1 (`Fournisseurs.ts`/`OrganisateursEvenements.ts`).
